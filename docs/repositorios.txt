Repositorio principal (proyecto completo):
- https://github.com/tu-usuario/rednorte-proyecto

Frontend React:
- https://github.com/tu-usuario/rednorte-frontend
  Contiene la interfaz React que consume el BFF y muestra pacientes y citas.

BFF Backend:
- https://github.com/tu-usuario/rednorte-bff
  Contiene el Backend For Frontend que unifica datos de `ms-pacientes` y `ms-citas`.

Microservicio Pacientes:
- https://github.com/tu-usuario/rednorte-ms-pacientes
  Servicio REST para gestión de pacientes.

Microservicio Citas:
- https://github.com/tu-usuario/rednorte-ms-citas
  Servicio REST para gestión de citas con Factory Method para tipos de cita.

Arquetipos Maven:
- https://github.com/tu-usuario/rednorte-arquetipos
  Contiene el arquetipo Maven base para generar nuevos microservicios backend.

> Nota: Actualizar estos enlaces con las URL reales de los repositorios en GitHub.
